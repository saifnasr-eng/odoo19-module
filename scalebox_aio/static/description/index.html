Scalebox ERP

لوحة تحكم متكاملة لإدارة أعمالك بكفاءة عالية

نظرة عامة على الموديول

يوفر موديول Scalebox ERP لوحة بيانات شاملة تعرض جميع المؤشرات المالية والتشغيلية في مكان واحد.

المميزات الرئيسية

  • لوحة بيانات رئيسية بكروت ملونة وتفاعلية
  • مبيعات اليوم ومبيعات الفترة مع عدد العمليات
  • مرتجعات المبيعات وصافي المبيعات
  • مبيعات نقاط البيع (اليوم والفترة)
  • مشتريات الفترة مع المرتجعات
  • مصروفات الفترة مع عدد السندات
  • رصيد الخزائن والبنوك
  • مديونية العملاء ومستحقات الموردين
  • فلترة بالتاريخ مع زر تحديث فوري
  • تقارير قابلة للطباعة وتصدير PDF
